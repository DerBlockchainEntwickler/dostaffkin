# DoStaffKin

Веб-приложение для заказа и отслеживания доставки посылок между странами и внутри стран.

## Функционал

- **Главная страница**: Навигация по приложению
- **Заказ доставки**: Форма с маршрутом (откуда/куда), размером посылки (XS-XL), скоростью доставки, личными данными (имя, телефон, комментарий). Интеграция с Yandex Maps для расчета маршрута
- **Отслеживание**: Поиск информации о доставке по номеру отправления

## Технологии

- **Frontend**: Angular 21, TypeScript
- **UI**: Reactive Forms, ngx-toastr для уведомлений
- **Карты**: Yandex Maps API
- **API**: Внешний сервис доставки (testologia.ru)
- **Сборка**: Angular CLI, Vite
- **Тестирование**: Vitest

## Установка и запуск
```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm start
# или
ng serve

# Сборка для production
npm run build
# или
ng build
```

Приложение будет доступно на `http://localhost:4200`.
